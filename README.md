# 20210107 本週作業：成績單 (transcript.cpp)

請寫一個使用 `vector<int>` 的程式，完成以下步驟的功能：

1. 讓使用者可以輸入多個成績到 scores 中，成績數量為偶數（為了第三步），最後輸入 0 停止輸入。

2. 排序

3. 透過複製分離 vector 的方式，計算出全部平均、前半同學平均與後半同學平均。

## 測試範例

### 輸入範例
```
100 40 80 20 0
```

### 輸出範例
```
平均：60
前半：90
後半：30
```
